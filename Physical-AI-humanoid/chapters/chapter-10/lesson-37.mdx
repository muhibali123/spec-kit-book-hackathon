## Key Concepts

- **Assistive Robotics**: Robots designed to assist individuals with disabilities or special needs
- **Healthcare Robotics**: Robots used in medical settings for patient care, therapy, or support
- **Human-Robot Interaction in Healthcare**: Special considerations for medical environments
- **Safety Standards**: Regulatory and safety requirements for medical robots
- **Therapeutic Robotics**: Robots used for rehabilitation and therapy purposes
- **Elderly Care**: Robotic assistance for aging populations
- **Compliance and Ethics**: Legal and ethical frameworks for healthcare robots
- **Social Acceptance**: Patient and caregiver acceptance of robotic assistance

## Theory Summary

Healthcare and assistive robotics represent one of the most promising applications for humanoid robots, addressing critical needs in an aging society and healthcare workforce shortages. These robots must operate in sensitive environments where safety, reliability, and appropriate social interaction are paramount.

The technical requirements for healthcare robots are particularly stringent. Unlike industrial robots that operate in controlled environments, healthcare robots must navigate complex, dynamic environments populated by vulnerable individuals. They must be able to adapt to unpredictable situations while maintaining the highest safety standards. This requires sophisticated perception systems, robust control algorithms, and fail-safe mechanisms.

Human-robot interaction in healthcare settings requires special consideration of psychological and social factors. Patients may experience anxiety or resistance when interacting with robots, especially if they are elderly or have cognitive impairments. Healthcare robots must be designed with intuitive interfaces, appropriate social behaviors, and the ability to build trust with users.

Safety standards for medical robots are governed by organizations such as the FDA in the United States and the European Medicines Agency in Europe. These standards require extensive testing and validation to ensure that robots do not pose risks to patients or healthcare workers. The standards cover mechanical safety, electrical safety, software reliability, and cybersecurity.

Therapeutic robotics represents a specialized application where humanoid robots assist in rehabilitation and therapy. These robots can provide consistent, repeatable therapy sessions, track patient progress, and adapt exercises to individual needs. The social presence of humanoid robots can also motivate patients to engage more actively in therapy.

Elderly care applications address the growing need for assistance as populations age globally. Healthcare robots can provide companionship, medication reminders, fall detection, and basic assistance with activities of daily living. The challenge lies in creating robots that can operate independently while maintaining appropriate levels of human oversight.

Compliance and ethics in healthcare robotics encompass issues such as patient privacy, informed consent, and the appropriate division of responsibilities between robots and human caregivers. These considerations must be addressed in both the design and deployment phases.

Social acceptance of healthcare robots varies significantly across cultures and demographics. Successful deployment requires understanding and addressing the concerns of patients, families, and healthcare professionals.

```python
import numpy as np
import matplotlib.pyplot as plt
from datetime import datetime, timedelta
from typing import Dict, List, Tuple, Optional
import random
import json
from dataclasses import dataclass, asdict
from enum import Enum

class PatientState(Enum):
    """Enumeration of possible patient states"""
    NORMAL = "normal"
    NEEDS_ASSISTANCE = "needs_assistance"
    EMERGENCY = "emergency"
    SLEEPING = "sleeping"
    EXERCISING = "exercising"

class InteractionType(Enum):
    """Types of interactions the robot can have"""
    GREETING = "greeting"
    MEDICATION_REMINDER = "medication_reminder"
    EXERCISE_GUIDANCE = "exercise_guidance"
    EMERGENCY_RESPONSE = "emergency_response"
    SOCIAL_INTERACTION = "social_interaction"

@dataclass
class Patient:
    """Represents a patient in the healthcare system"""
    id: str
    name: str
    age: int
    conditions: List[str]
    medication_schedule: Dict[str, List[str]]  # med_name -> [times]
    exercise_routine: List[Dict]  # List of exercise activities
    preferences: Dict[str, str]  # Patient preferences
    last_seen: datetime
    state: PatientState
    vital_signs: Dict[str, float]  # Heart rate, blood pressure, etc.

class HealthcareRobot:
    """Healthcare robot that provides assistance and monitoring"""

    def __init__(self, name: str = "CareBot", max_patients: int = 10):
        self.name = name
        self.max_patients = max_patients
        self.patients: Dict[str, Patient] = {}
        self.interaction_history: List[Dict] = []
        self.current_time = datetime.now()
        self.battery_level = 100.0
        self.location = (0, 0)  # Current location in facility
        self.emergency_thresholds = {
            'heart_rate': (60, 100),
            'blood_pressure_systolic': (100, 140),
            'blood_pressure_diastolic': (60, 90),
            'temperature': (36.0, 37.5)
        }

    def register_patient(self, patient: Patient) -> bool:
        """Register a new patient with the robot"""
        if len(self.patients) >= self.max_patients:
            print(f"Cannot register patient: Maximum capacity ({self.max_patients}) reached")
            return False

        self.patients[patient.id] = patient
        print(f"Patient {patient.name} registered successfully")
        return True

    def check_medication_schedule(self) -> List[Tuple[str, str, str]]:
        """Check for scheduled medications and return list of reminders needed"""
        reminders = []

        for patient_id, patient in self.patients.items():
            for med, schedule_times in patient.medication_schedule.items():
                for schedule_time in schedule_times:
                    # Check if it's time for medication
                    if self._is_time_for_medication(schedule_time):
                        reminders.append((patient_id, med, schedule_time))

        return reminders

    def _is_time_for_medication(self, schedule_time: str) -> bool:
        """Check if it's time for a medication based on schedule time"""
        # Simple time comparison - in a real system, this would be more sophisticated
        current_hour = self.current_time.hour
        current_minute = self.current_time.minute

        # Parse schedule time (HH:MM format)
        try:
            hour, minute = map(int, schedule_time.split(':'))
            # Allow 15-minute window for medication
            time_diff = abs((current_hour * 60 + current_minute) - (hour * 60 + minute))
            return time_diff <= 15
        except:
            return False

    def monitor_patients(self) -> List[str]:
        """Monitor all patients and detect any that need assistance or are in emergency"""
        alerts = []

        for patient_id, patient in self.patients.items():
            # Check vital signs for abnormalities
            if self._check_vital_signs(patient):
                # Update patient state to emergency
                patient.state = PatientState.EMERGENCY
                alerts.append(f"EMERGENCY: Patient {patient.name} vital signs critical")

            # Check for other conditions
            elif self._check_patient_behavior(patient):
                patient.state = PatientState.NEEDS_ASSISTANCE
                alerts.append(f"ASSISTANCE NEEDED: Patient {patient.name} requires help")

        return alerts

    def _check_vital_signs(self, patient: Patient) -> bool:
        """Check if patient's vital signs are outside normal ranges"""
        vs = patient.vital_signs

        # Check each vital sign against thresholds
        for vital, value in vs.items():
            if vital in self.emergency_thresholds:
                min_val, max_val = self.emergency_thresholds[vital]
                if value < min_val or value > max_val:
                    return True
        return False

    def _check_patient_behavior(self, patient: Patient) -> bool:
        """Check for behavioral indicators that require assistance"""
        # Simple check: if patient hasn't been active for a long time
        time_since_seen = self.current_time - patient.last_seen
        if time_since_seen > timedelta(minutes=30):
            return True

        # Check if patient is in a potentially problematic state
        if patient.state == PatientState.EMERGENCY:
            return True

        return False

    def generate_interaction(self, patient_id: str, interaction_type: InteractionType) -> str:
        """Generate appropriate interaction for the patient"""
        if patient_id not in self.patients:
            return f"Patient {patient_id} not found"

        patient = self.patients[patient_id]

        interaction_content = ""

        if interaction_type == InteractionType.GREETING:
            time_of_day = "morning" if self.current_time.hour < 12 else "afternoon" if self.current_time.hour < 18 else "evening"
            interaction_content = f"Good {time_of_day}, {patient.name}! I'm {self.name}, your healthcare assistant. How are you feeling today?"

        elif interaction_type == InteractionType.MEDICATION_REMINDER:
            # Find current medications due
            due_meds = []
            for med, schedule in patient.medication_schedule.items():
                for time_str in schedule:
                    if self._is_time_for_medication(time_str):
                        due_meds.append(med)

            if due_meds:
                med_list = ", ".join(due_meds)
                interaction_content = f"Hi {patient.name}, it's time for your medication: {med_list}. Please take them with water."
            else:
                interaction_content = f"Hello {patient.name}, just checking if you've taken your medications today."

        elif interaction_type == InteractionType.EXERCISE_GUIDANCE:
            if patient.exercise_routine:
                exercise = random.choice(patient.exercise_routine)
                interaction_content = f"{patient.name}, it's time for your exercise: {exercise['name']}. {exercise['instructions']}"
            else:
                interaction_content = f"{patient.name}, would you like to do some gentle exercises today?"

        elif interaction_type == InteractionType.EMERGENCY_RESPONSE:
            interaction_content = f"EMERGENCY ALERT: {patient.name}, help is on the way. Please stay calm and remain where you are."

        elif interaction_type == InteractionType.SOCIAL_INTERACTION:
            # Generate social interaction based on patient preferences
            if 'conversation_topic' in patient.preferences:
                topic = patient.preferences['conversation_topic']
                interaction_content = f"{patient.name}, would you like to talk about {topic}? I'd love to hear your thoughts."
            else:
                interaction_content = f"Hello {patient.name}, how about we chat for a moment? Is there anything on your mind?"

        # Log the interaction
        interaction_log = {
            'timestamp': self.current_time.isoformat(),
            'patient_id': patient_id,
            'interaction_type': interaction_type.value,
            'content': interaction_content,
            'location': self.location
        }
        self.interaction_history.append(interaction_log)

        return interaction_content

    def update_patient_state(self, patient_id: str, new_state: PatientState,
                           vital_signs: Optional[Dict] = None) -> bool:
        """Update patient state and vital signs"""
        if patient_id not in self.patients:
            return False

        patient = self.patients[patient_id]
        patient.state = new_state
        patient.last_seen = self.current_time

        if vital_signs:
            patient.vital_signs.update(vital_signs)

        return True

    def get_patient_summary(self, patient_id: str) -> Dict:
        """Get a summary of patient information"""
        if patient_id not in self.patients:
            return {}

        patient = self.patients[patient_id]
        return {
            'name': patient.name,
            'state': patient.state.value,
            'last_seen': patient.last_seen.isoformat(),
            'vital_signs': patient.vital_signs,
            'conditions': patient.conditions,
            'next_medication': self._get_next_medication_time(patient)
        }

    def _get_next_medication_time(self, patient: Patient) -> str:
        """Get the next scheduled medication time for a patient"""
        # Simple implementation - find the next medication time
        current_minutes = self.current_time.hour * 60 + self.current_time.minute

        for med, schedule in patient.medication_schedule.items():
            for time_str in schedule:
                try:
                    hour, minute = map(int, time_str.split(':'))
                    med_minutes = hour * 60 + minute
                    if med_minutes > current_minutes:
                        return f"{med} at {time_str}"
                except:
                    continue

        return "No medications scheduled soon"

def simulate_healthcare_robot():
    """Simulate a healthcare robot in action"""
    print("Healthcare Robot Simulation")
    print("=" * 30)

    # Create healthcare robot
    robot = HealthcareRobot(name="CareBot-01")

    # Create sample patients
    patient1 = Patient(
        id="P001",
        name="Alice Johnson",
        age=78,
        conditions=["Hypertension", "Diabetes"],
        medication_schedule={
            "Blood Pressure Med": ["08:00", "20:00"],
            "Diabetes Med": ["09:00", "13:00", "17:00"]
        },
        exercise_routine=[
            {"name": "Arm stretches", "instructions": "Gently stretch your arms 10 times"},
            {"name": "Ankle rotations", "instructions": "Rotate your ankles 10 times each direction"}
        ],
        preferences={"conversation_topic": "gardening"},
        last_seen=datetime.now() - timedelta(minutes=5),
        state=PatientState.NORMAL,
        vital_signs={"heart_rate": 72, "blood_pressure_systolic": 125,
                   "blood_pressure_diastolic": 80, "temperature": 36.7}
    )

    patient2 = Patient(
        id="P002",
        name="Robert Chen",
        age=82,
        conditions=["Arthritis", "Heart Condition"],
        medication_schedule={
            "Heart Med": ["08:30", "20:30"],
            "Pain Med": ["10:00", "15:00"]
        },
        exercise_routine=[
            {"name": "Breathing exercises", "instructions": "Take 5 deep breaths slowly"},
            {"name": "Seated march", "instructions": "March in place while seated for 30 seconds"}
        ],
        preferences={"conversation_topic": "travel"},
        last_seen=datetime.now() - timedelta(minutes=10),
        state=PatientState.NORMAL,
        vital_signs={"heart_rate": 68, "blood_pressure_systolic": 130,
                   "blood_pressure_diastolic": 78, "temperature": 36.5}
    )

    # Register patients
    robot.register_patient(patient1)
    robot.register_patient(patient2)

    # Simulate robot operations over time
    print("\nSimulating robot operations...")

    # Check for medication reminders
    print("\nChecking medication schedule:")
    reminders = robot.check_medication_schedule()
    for patient_id, med, time in reminders:
        patient_name = robot.patients[patient_id].name
        print(f"  Reminder: {patient_name} needs {med} at {time}")
        # Generate medication reminder interaction
        interaction = robot.generate_interaction(patient_id, InteractionType.MEDICATION_REMINDER)
        print(f"  Robot says: {interaction}")

    # Monitor patients for any issues
    print("\nMonitoring patients:")
    alerts = robot.monitor_patients()
    for alert in alerts:
        print(f"  {alert}")

    # Generate some interactions
    print("\nGenerating interactions:")
    for patient_id in robot.patients.keys():
        # Greeting interaction
        greeting = robot.generate_interaction(patient_id, InteractionType.GREETING)
        print(f"  Greeting: {greeting}")

        # Social interaction
        social = robot.generate_interaction(patient_id, InteractionType.SOCIAL_INTERACTION)
        print(f"  Social: {social}")

    # Simulate an emergency scenario
    print("\nSimulating emergency scenario for patient 1:")
    robot.update_patient_state("P001", PatientState.EMERGENCY,
                              {"heart_rate": 120, "blood_pressure_systolic": 160})

    # Check for alerts again
    alerts = robot.monitor_patients()
    for alert in alerts:
        print(f"  {alert}")

    # Emergency response
    emergency_response = robot.generate_interaction("P001", InteractionType.EMERGENCY_RESPONSE)
    print(f"  Emergency response: {emergency_response}")

    # Print patient summaries
    print("\nPatient summaries:")
    for patient_id in robot.patients.keys():
        summary = robot.get_patient_summary(patient_id)
        print(f"  {summary['name']}: State={summary['state']}, Next med={summary['next_medication']}")

    print(f"\nTotal interactions logged: {len(robot.interaction_history)}")
    print("Healthcare robot simulation completed!")

# Run the simulation
simulate_healthcare_robot()
```
